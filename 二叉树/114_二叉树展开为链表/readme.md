给你二叉树的根结点 root ，请你将它展开为一个单链表：

展开后的单链表应该同样使用 TreeNode ，其中 right 子指针指向链表中下一个结点，而左子指针始终为 null 。
展开后的单链表应该与二叉树 先序遍历 顺序相同。
 

示例 1：


输入：root = [1,2,5,3,4,null,6]
输出：[1,null,2,null,3,null,4,null,5,null,6]
示例 2：

输入：root = []
输出：[]
示例 3：

输入：root = [0]
输出：[0]
 

提示：

树中结点数在范围 [0, 2000] 内
-100 <= Node.val <= 100
 

进阶：你可以使用原地算法（O(1) 额外空间）展开这棵树吗？

解题思路：
- 思路来源 ：目标链表顺序要等于二叉树的先序遍历（根→左→右）。所以对每个节点 cur 来说，应该把它的“左子树整体”插到 cur 和 cur->right 之间。
- 怎么“插进去” ：
- 如果 cur->left 不为空：
  1. 在 cur->left 这棵子树里，找到最右边的节点 pred （一直沿着 right 走到尽头）。
  2. 把原来的右子树接到 pred->right ： pred->right = cur->right
  3. 把左子树挪到右边： cur->right = cur->left
  4. 断开左边： cur->left = nullptr
- 然后 cur = cur->right 继续处理下一个节点
- 这套操作每次都把当前节点的左边清空，并把“先序里应该紧跟在根后面的左子树”移动到右链上，所以最终得到的就是先序顺序的单链表。
