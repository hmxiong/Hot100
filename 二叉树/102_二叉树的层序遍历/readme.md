给你二叉树的根节点 root ，返回其节点值的 层序遍历 。 （即逐层地，从左到右访问所有节点）。

 

示例 1：


输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
示例 2：

输入：root = [1]
输出：[[1]]
示例 3：

输入：root = []
输出：[]
 

提示：

树中节点数目在范围 [0, 2000] 内
-1000 <= Node.val <= 1000

解题思路：
- “层序遍历”= 按层从左到右访问，本质就是图/树的 BFS（广度优先搜索）。
- BFS 的天然工具是队列 queue ：
- 先把根节点入队
- 每次取出当前队列里的一整层（用 sz = q.size() 记住本层节点数）
- 依次弹出这 sz 个节点，把它们的左右孩子（如果存在）入队
- 这样每个节点进队出队一次：时间 O(n) ，空间 O(n) （队列最多存一层节点）。
