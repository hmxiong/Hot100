给你链表的头结点 head ，请将其按 升序 排列并返回 排序后的链表 。

 

示例 1：


输入：head = [4,2,1,3]
输出：[1,2,3,4]
示例 2：


输入：head = [-1,5,3,4,0]
输出：[-1,0,3,4,5]
示例 3：

输入：head = []
输出：[]
 

提示：

链表中节点的数目在范围 [0, 5 * 104] 内
-105 <= Node.val <= 105
 

进阶：你可以在 O(n log n) 时间复杂度和常数级空间复杂度下，对链表进行排序吗？

解题思路：- 链表排序最常用、也最适配的算法是“归并排序”：
  - 数组常用快排，但链表不支持随机访问（不能 O(1) 取中点、分区也不方便）。
  - 归并排序只依赖“拆分 + 合并有序链表”，这些操作在链表上非常自然。
- 进阶要求“常数级额外空间”：用 自底向上（bottom-up）的迭代归并排序 ，避免递归栈空间。
- 核心流程：
  - step=1：两两合并长度为 1 的段
  - step=2：合并长度为 2 的段
  - step=4、8…直到 step >= n
  - 每次：把链表切成 left(长度step) 、 right(长度step) 两段，合并后接回去
时间复杂度 O(n log n) ，额外空间 O(1) （不算返回值）。
