# LeetCode Hot 100（C++ 学习记录）

这个仓库用于我个人学习 LeetCode Hot 100 题目，主要目标是把每道题的核心思路、代码实现、以及自测方式沉淀下来，便于后续复习和持续迭代。

## 目录结构

- 每道题一个文件夹：`题号_题目名/`
  - `readme.md`：题意 + 核心思路（学习笔记）
  - `test_*.cpp`：可独立编译运行的自测代码（题解 + 样例 + 随机对拍/暴力校验）

示例：

- `1_两数之和/readme.md`
- `438_找到字符串中所有字母异位词/test_find_anagrams.cpp`

## 运行自测

大部分题目我会提供一个 `test_*.cpp` 的自测文件，推荐使用 `g++` 直接编译运行：

```bash
g++ -std=c++17 -O2 -Wall -Wextra "题号_题目名/test_xxx.cpp" -o "题号_题目名/test_xxx" && "题号_题目名/test_xxx"
```

运行通过会输出：

```text
All tests passed.
```

## 题解思路约定

- 优先选择时间复杂度更优的经典解法（如哈希、双指针、滑动窗口、单调队列、前缀和等）
- 若题目有“进阶/常量空间”要求，尽量给出满足进阶的实现
- 自测优先使用：
  - 题目样例 + 边界用例
  - 小规模随机数据对拍（用暴力/参考实现做校验）

## 提交到 GitHub 的建议

- 只提交 `*.cpp`、`*.md` 等源码与笔记文件
- 本地编译生成的可执行文件（例如 `test_xxx`）建议通过 `.gitignore` 忽略
